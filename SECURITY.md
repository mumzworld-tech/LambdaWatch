# Security Policy

## Supported Versions

Only the **latest major release branch** of `mumzworld-tech/LambdaWatch` receives official security updates and patches. We encourage all users to upgrade to the latest stable version as soon as possible to ensure the highest level of security.

| Version | Supported |
| :--- | :--- |
| Latest Major Release | ✅ Yes |
| Previous Major Releases | ❌ No |

## Reporting a Vulnerability

We take security vulnerabilities seriously and appreciate the community's help in responsibly identifying and reporting issues.

**Please do not open a public GitHub Issue for security concerns.**

Instead, please report the vulnerability immediately and privately using one of the following methods:

1.  **Email:** Send an email to **tech.sre@mumzworld.com**.
2.  **GitHub Private Reporting:** Use the official GitHub feature to [privately report a security vulnerability](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability).

### How to Report

In your report, please include:

* A clear description of the vulnerability.
* The package version(s) affected.
* Detailed steps to reproduce the vulnerability (including any necessary code snippets or configuration).
* The potential impact of the vulnerability.

### Our Policy

* We will acknowledge your report as soon as possible.
* We will keep you informed of the remediation process.
* We will implement a fix and release a patched version as quickly as possible under a Coordinated Vulnerability Disclosure (CVD) policy.
* We request that you keep the vulnerability confidential until a patch is released and announced.
